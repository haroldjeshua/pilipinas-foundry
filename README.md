# tipong-pinoy

Filipino Free Font Library — a design-engineering initiative for previewing
free/open-license fonts made by Filipino type designers.

## Development

```sh
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev` — start dev server
- `pnpm build` — typecheck + production build
- `pnpm lint` — run ESLint
- `pnpm format` — format with Prettier
- `pnpm typecheck` — TypeScript type checking
